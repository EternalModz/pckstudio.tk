PCK INSTALLATION


Wii U
Installing custom pck files on Wii U is probably the easiest amongst all other consoles. There's the Temporary install method which streams the file to your console as you play, then there's the perma install method which permanently installs the pck to your console. Both methods require an already existing pck to replaced.

Ps3
Installing custom pck files on Ps3 is the second easiest, most overall the most convenient. Custom skins can be installed on both OFW and CFW consoles. Unlike Wii U, on Ps3, you don't have to replace an already existing pck(although you still can). Putting the pck file in a new folder in the DLC directory will add the pck file to the game as a completely new Skin Pack.

Xbox 360
Installing custom pck files on Xbox 360 although possible, requires a hardware modification of your console. I only do tutorials and showcases of soft-mods because they are the most convenient for all users, so I sadly do not have any tutorials on installing to your Xbox 360. If you have a modded console, you can install the pck by replacing an already existing pck or adding a new folder to the DLC directory just like on other consoles.

FAQ

This error is happening because a piece of metadata is in use that hasn't been added to the main metadatabase (File>Meta). This usually happens when custom models are created for skins because "OFFSET" isn't a default metadatabase entry. Just go to to File>Meta and right click in the window that appears, then choose "Add", then add the piece of metadata you used that isn't in this list. Example: File>Meta, Right Click>Add, "OFFSET", Add. Upon adding all metadata thats in use, you'll be able to successfully save your pck.

Sometimes 4j Studios updates data in their pck files, so some updates will overwrite pck files. Don't worry, all you have to do is reinstall the pck when this happens.

Although all pck files are in the same format, they all have different structures. Some structures are a little more advanced because of some extra data 4j Studios added. As of now, it still hasn't been figured out. An example of one of these pcks is the Story Mode pck. Don't worry, you can just create a new pck (File>New) if you need a template. This could also mean you're trying to open a corrupted pck.
